Functionality and Purpose of this File

This file contains functionality that is shared between several types of scene elements, including entity elements, animated entity elements, emissive entity elements, group elements, and point light elements. Note that some of the functionality in this file is only used by some of the scene elements mentioned, for example the function add_emissive_material_imgui_edit_section is only used by emissive entity elements, and the function add_material_imgui_edit_section is only used by entity elements and animated entity elements.


(Important) Method Documentation


void EditorScene::LocalTransformComponent::add_local_transform_imgui_edit_section(MasterRenderScene& /*render_scene*/, const SceneContext& scene_context)

This function adds the ImGui elements that allow the user to interact with the entity's position, rotation, and scale via the UI. It is called every frame.

Parameters

MasterRenderScene& /*render_scene*/: The master render scene, defined in MasterRenderScene.h, which holds various information relevant to the scene. Not used in this function.

const SceneContext& scene_context: An object (defined in SceneContext.h and created in main.cpp) that holds information on the window, window manager, model loader, texture loader, and a bool that indicates if ImGui elements are enabled.

Where is this called?

This is called inside the add_imgui_edit_section function of AnimatedEntityElement.cpp, EmissiveEntityElement.cpp, EntityElement.cpp, and GroupElement.cpp.




void EditorScene::LitMaterialComponent::add_material_imgui_edit_section(MasterRenderScene& /*render_scene*/, const SceneContext& /*scene_context*/)

In the base project this function has no functionality, however in a finished project it should add the ImGui elements that allow the user to interact with the material properties of entities and animated entities. It is called every frame.

Parameters

MasterRenderScene& /*render_scene*/: The master render scene, defined in MasterRenderScene.h, which holds various information relevant to the scene. Not used in this function.

const SceneContext& /*scene_context*/: An object (defined in SceneContext.h and created in main.cpp) that holds information on the window, window manager, model loader, texture loader, and a bool that indicates if ImGui elements are enabled. Not used in this function.

Remarks

Note that update_instance_data is only called if material_changed is true. Also note that most ImGui functions return a bool that indicates if their UI element has been interacted with since the last frame.

Where is this called?

This is called inside the add_imgui_edit_section function of AnimatedEntityElement.cpp and EntityElement.cpp.
