res/shaders/emissive_entity/frag.glsl


Functionality and Purpose of this File

This file is the fragment shader used to render emissive entities.


Method Documentation


void main()

This function sets the value for out_colour, which corresponds to the fragment colour, based on texture_colour and emissive_tint.

Remarks

For more information on what texture(emissive_texture, frag_in.texture_coordinate).rgb does, see the documentation here. Also notice that unlike in the fragment shaders for entity and animated_entity, resolve_textured_light_calculation is not used here. This is because the emissive entity is not a lit entity, and so its colour is only dependent upon its texture and emissive tint.
