src

main.cpp

Functionality and Purpose of this File

This file can be broken down into three separate sections - before, inside, and after the main render loop. Before the main render loop a variety of important objects of different classes are initialized, effectively setting up the project. Inside the main render loop every iteration renders a frame (an overview of how this is done can be found here), and this will run until the project is closed. After the main render loop, cleanup is performed and any objects previously created are destroyed.

src/rendering/renders/shaders

BaseEntityShader.cpp

Functionality and Purpose of this File

This file contains entity shading functions used by all entity types, such as set_instance_data and set_global_data which pass the model_matrix and projection_view_matrix of each entity to the shaders.

Header file for BaseEntityShader.cpp, this file declares the functions associated with the base entity shader.


BaseLitEntityShader.cpp

Functionality and Purpose of this File

This file's functionality is an extension of BaseEntityShader.cpp and includes a variety of functions that get and send light- and material-related information to the shaders.


BaseLitEntityShader.h

Functionality and Purpose of this File

Header file for BaseLitEntityShader.cpp, this file declares the structs, variables, and functions associated with the base lit entity shader. In particular, the BaseLitEntityMaterial struct which holds each lit entity's material properties is declared in this file.


ShaderInterface.cpp

Functionality and Purpose of this File

This file contains the functionality associated with the helper class ShaderInterface, which acts as an interface for the shaders. It is used in BaseEntityShader.cpp, which is in turn used in AnimatedEntityRenderer.cpp, EmissiveEntityRenderer.cpp, and EntityRenderer.cpp.


ShaderInterface.h

Functionality and Purpose of this File

src/rendering/scene

Animator.cpp

Functionality and Purpose of this File

This file defines the animate function used to control the animation of animated entities.


Animator.h

Functionality and Purpose of this File

Header file for Animator.cpp, this file declares and defines the AnimationParameters struct as well as a variety of functions used to control the animation of animated entities.


GlobalData.h

Functionality and Purpose of this File

This file declares the base class for all GlobalData structs which contain camera data, to provide a common interface for applying a camera state. The member variables of the GlobalData struct are most notably used in the set_global_data function of BaseEntityShader.cpp.


Lights.cpp

Functionality and Purpose of this File

In this file, the functions get_nearest_point_lights and get_nearest_lights are defined. From a high level overview, these functions are used in the render functions of EntityRenderer.cpp and AnimatedEntityRenderer.cpp, where they are used to determine the subset of point lights we provide information about to the shaders.


Lights.h

Functionality and Purpose of this File

Header file for Lights.cpp. In this file two structs are declared: PointLight and LightScene. The PointLight struct is a representation of a point light scene element, and the LightScene struct contains a set with all point lights and a helper function to get the nearest point lights up to a specified max_count.


MasterRenderScene.cpp

Functionality and Purpose of this File

This file contains the functionality associated with the master render scene, which holds a copy of each renderer's render scene, a copy of the light scene, and the animator used to control animation. The master render scene is primarily used as an interface for adding and removing entities and lights.


MasterRenderScene.h

Functionality and Purpose of this File

Header file for MasterRenderScene.cpp, this file declares the functions associated with the master render scene.


RenderedEntity.h

Functionality and Purpose of this File

This file defines the RenderedEntity and AnimatedRenderedEntity structs used by the various entity renderers.


RenderScene.h

Functionality and Purpose of this File

This file defines the RenderScene struct used by the various entity renderers, which holds entity and global data information.

src/scene/editor_scene

AnimatedEntityElement.cpp

Functionality and Purpose of this File

This file contains the functionality associated with animated entities. Animated entities are just like normal entities, except they can have animations applied to them. Some notable functions include new_default which defines the default properties for UI-created animated entities, and add_imgui_edit_section which defines ImGui elements that allow the user to manipulate the animated entities in a scene.


AnimatedEntityElement.h

Functionality and Purpose of this File

Header file for AnimatedEntityElement.cpp, this file declares the member variables and functions associated with animated entities.


EmissiveEntityElement.cpp

Functionality and Purpose of this File

This file contains the functionality associated with emissive entities. Emissive entities are (externally) unlit entities whose colour is only dependent upon their applied texture and emissive tint (that is, they are not affected by the lights in the scene). Some notable functions include new_default which defines the default properties for UI-created emissive entities, and add_imgui_edit_section which defines ImGui elements that allow the user to manipulate the emissive entities in a scene.


EmissiveEntityElement.h

Functionality and Purpose of this File

Header file for EmissiveEntityElement.cpp, this file declares the member variables and functions associated with emissive entities.


EntityElement.cpp

Functionality and Purpose of this File

This file contains the functionality associated with normal entities. Some notable functions include new_default which defines the default properties for UI-created normal entities, and add_imgui_edit_section which defines ImGui elements that allow the user to manipulate the normal entities in a scene.


EntityElement.h

Functionality and Purpose of this File

Header file for EntityElement.cpp, this file declares the member variables and functions associated with normal entities.


GroupElement.cpp

Functionality and Purpose of this File

This file contains the functionality associated with group scene elements. Group scene elements are used to group multiple scene elements, allowing the user to manipulate them together. The most notable function is update_instance_data which is how the group element updates the transforms of its constituent scene elements.


GroupElement.h

Functionality and Purpose of this File

Header file for GroupElement.cpp, this file declares the member variables and functions associated with group scene elements.


PointLightElement.cpp

Functionality and Purpose of this File

This file contains all of the functions associated with the point light scene element. This includes the constructor, json-related functions, ImGui elements, update instance data function, and an accessor function for the light's element type name.


PointLightElement.h

Functionality and Purpose of this File

Header file for PointLightElement.cpp, this file declares the member variables and functions associated with point light scene elements.


SceneElement.cpp

Functionality and Purpose of this File

This file contains functionality that is shared between several types of scene elements, including entity elements, animated entity elements, emissive entity elements, group elements, and point light elements. Note that some of the functionality in this file is only used by some of the scene elements mentioned, for example the function add_emissive_material_imgui_edit_section is only used by emissive entity elements, and the function add_material_imgui_edit_section is only used by entity elements and animated entity elements.


SceneElement.h

Functionality and Purpose of this File

Header file for SceneElement.cpp, this file declares the member variables and functions associated with the scene elements.
