vert.glsl (emissive entity)


Location in Project

res/shaders/emissive_entity/vert.glsl


Functionality and Purpose of this File

This file is the vertex shader used to render emissive entities.


Method Documentation


void main()

This function simply calculates and assigns the values for ws_position (world space vertex position) and texture_coordinate, which are members of the vertex_out interface block. It also sets the value for gl_Position, which is the result of applying the projection view matrix to the world space vertex position.

Remarks

Unlike the vertex shaders for entity and animated entity, this shader doesn't get or use any light-related data. This is because the emissive entity is an (externally) unlit entity and it's colour is only dependent upon it's applied texture and emissive tint.
