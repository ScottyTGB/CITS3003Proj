res/shaders/animated_entity/vert.glsl


Functionality and Purpose of this File

This file is the vertex shader used to render animated entities.


Method Documentation


void main()

This function assigns the values for lighting_result and texture_coordinate, both of which are members of the vertex_out interface block. It also sets the value for gl_Position, which is the result of applying the projection view matrix to the world space vertex position.

Remarks

The total_light_calculation function used to determine the value of vertex_out.lighting_result is defined in lights.glsl, as are the LightCalculatioData and Material structs. Overall, this vertex shader is very similar to the vertex shader used for normal entities, with the only difference being that ws_position and ws_normal (and hence vertex_out.lighting_result) are calculated based on the bone_transform matrix.
