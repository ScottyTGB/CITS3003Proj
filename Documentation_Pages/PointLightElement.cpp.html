src/scene/editor_scene/PointLightElement.cpp


Functionality and Purpose of this File

This file contains all of the functions associated with the point light scene element. This includes the constructor, json-related functions, ImGui elements, update instance data function, and an accessor function for the light's element type name.


Method Documentation


std::unique_ptr<EditorScene::PointLightElement> EditorScene::PointLightElement::new_default(const SceneContext& scene_context, EditorScene::ElementRef parent)

This is the constructor for the point light scene element.

Parameters

const SceneContext& scene_context: An object (defined in SceneContext.h and created in main.cpp) that holds information on the window, window manager, model loader, texture loader, and a bool that indicates if ImGui elements are enabled.

EditorScene::ElementRef parent: The parent element of the new point light. A parent element's transformation is applied to the transformation of their child elements.

Returns

A smart pointer to a new point light object.

Where is this called?

This is called in from_json defined in PointLightElement.cpp, and in the open function defined in EditorScene.cpp




std::unique_ptr<EditorScene::PointLightElement> EditorScene::PointLightElement::from_json(const SceneContext& scene_context, EditorScene::ElementRef parent, const json& j)

This function takes a json element containing values for the point light element's attributes and creates a new point light element with attributes matching the specified values.

Parameters

const SceneContext& scene_context: An object (defined in SceneContext.h and created in main.cpp) that holds information on the window, window manager, model loader, texture loader, and a bool that indicates if ImGui elements are enabled.

EditorScene::ElementRef parent: The parent element of the new point light. A parent element's transformation is applied to the transformation of their child elements.

const json& j: The json element containing the attribute values of the new point light object.

Returns

A smart pointer to a new point light object.

Where is this called?

This is called in the open function defined in EditorScene.cpp.

void EditorScene::PointLightElement::add_imgui_edit_section(MasterRenderScene& render_scene, const SceneContext& scene_context)

This function adds the ImGui elements that allow the user to interact with the point light's properties via the UI. It is called every frame and allows the user to modify the point light's position, colour, intensity, visual scale and if the sphere representing the light is visible.

Parameters

MasterRenderScene& render_scene: The master render scene, defined in MasterRenderScene.h, which holds various information relevant to the scene.

const SceneContext& scene_context: An object (defined in SceneContext.h and created in main.cpp) that holds information on the window, window manager, model loader, texture loader, and a bool that indicates if ImGui elements are enabled.

Where is this called?

This is called in the add_imgui_selection_editor function defined in EditorScene.cpp.




void EditorScene::PointLightElement::update_instance_data()

This function updates the point light element's instance data, in particular this involves updating the position of the point light element's light, and both the model matrix and emission tint of the point light element's sphere.

Where is this called?

This is called in a variety of places, however it is primarily called in add_imgui_edit_section, defined in PointLightElement.cpp.

Remarks

We don't update the colour of the point light element's light here because that is changed directly via the ImGui elements, defined in the add_imgui_edit_section.
